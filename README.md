# 🩺 AbdoAI

AI-powered assistant for abdominal surgery.

AbdoAI is a prototype AI assistant built with Python, Streamlit and RAG.  
The goal is to make surgical knowledge easier to access through a simple chat interface.

## Features

- GPT-powered chat
- Streamlit web interface
- Markdown-based knowledge base
- RAG architecture
- ChromaDB support planned
- Source citations planned

## Tech Stack

- Python
- Streamlit
- OpenAI API
- Markdown
- ChromaDB

## Roadmap

- ✅ Sprint 1 – Foundation
- ⬜ Multi-document RAG
- ⬜ Source citations
- ⬜ Medical illustrations
- ⬜ Voice support
